## 练习A：显示当前目录下的文件名（Linux应用程序）

### 题目要求
实现一个Linux应用程序，显示当前目录下的文件名。

### 实现方案
使用Rust标准库实现，类似Linux的`ls`命令。

### 代码说明

```rust
use std::fs;
use std::path::Path;

fn main() {
    // 获取当前目录路径
    let current_dir = Path::new(".");

    println!("当前目录下的文件：");

    // 读取并遍历当前目录
    match fs::read_dir(current_dir) {
        Ok(entries) => {
            for entry in entries {
                match entry {
                    Ok(entry) => {
                        // 获取文件名并打印
                        let file_name = entry.file_name();
                        println!("{}", file_name.to_string_lossy());
                    }
                    Err(e) => {
                        eprintln!("读取条目失败: {}", e);
                    }
                }
            }
        }
        Err(e) => {
            eprintln!("无法读取当前目录: {}", e);
        }
    }
}
```

### 关键API
- `Path::new(".")`: 创建路径对象
- `fs::read_dir()`: 读取目录条目
- `entry.file_name()`: 获取文件名
- `to_string_lossy()`: 安全转换为字符串

### 编译和运行

- 显示文件大小

### C语言版本（参考）

```c
#include <stdio.h>
#include <dirent.h>
#include <errno.h>

int main() {
    DIR *dir;
    struct dirent *entry;

    printf("当前目录下的文件：\n");

    dir = opendir(".");
    if (dir == NULL) {
        perror("无法打开当前目录");
        return 1;
    }

    while ((entry = readdir(dir)) != NULL) {
        printf("%s\n", entry->d_name);
    }

    closedir(dir);
    return 0;
}
```

## 练习B：打印调用栈信息（Linux应用程序）

### 题目要求
实现一个Linux应用程序，能打印出调用栈链信息。

### 实现方案
使用Rust的`std::backtrace`模块获取完整调用栈。

### 代码说明

```rust
use std::backtrace;

fn main() {
    // 设置启用backtrace
    std::env::set_var("RUST_BACKTRACE", "1");

    println!("调用栈信息演示：\n");

    // 创建多层调用
    first_function();
}

fn first_function() {
    println!("在 first_function() 中");
    second_function();
}

fn second_function() {
    println!("在 second_function() 中");
    third_function();
}

fn third_function() {
    println!("在 third_function() 中");

    // 方法1: 使用std::backtrace获取调用栈
    println!("\n=== 方法1: 使用std::backtrace ===");
    print_backtrace();

    // 方法2: 手动模拟简单的调用栈追踪
    println!("\n=== 方法2: 手动追踪调用链 ===");
    println!("当前调用链:");
    println!("  main()");
    println!("    └─> first_function()");
    println!("        └─> second_function()");
    println!("            └─> third_function()");
}

fn print_backtrace() {
    // 使用std::backtrace模块（Rust 1.65+）
    let bt = backtrace::Backtrace::new();
    println!("调用栈:\n{}", bt);
}
```

### 关键API
- `std::backtrace::Backtrace::new()`: 捕获当前调用栈
- `RUST_BACKTRACE`环境变量: 控制backtrace详细程度

### 编译和运行

```bash
# 编译
rustc exercise_b_backtrace.rs -o exercise_b_backtrace

# 运行（默认情况）
./exercise_b_backtrace

# 运行（完整backtrace）
RUST_BACKTRACE=full ./exercise_b_backtrace

# 运行（简短backtrace）
RUST_BACKTRACE=1 ./exercise_b_backtrace
```

## 练习C：在rCore中使用sleep系统调用

### 题目要求
实现一个基于rcore/ucore tutorial的应用程序，使用sleep系统调用睡眠5秒。

### 实现方案
这是rCore用户态应用程序，需要使用rCore的用户库。

### 代码说明

```rust
#![no_std]
#![no_main]

use user_lib::*;

#[no_mangle]
pub fn main() -> i32 {
    println!("Exercise C: Sleep system call test");
    println!("====================================");

    // 打印开始信息
    println!("开始睡眠...");

    // 使用sleep系统调用睡眠5秒
    // 注意：具体API取决于rCore教程版本
    sleep(5);

    println!("睡眠结束！已睡眠5秒");

    println!("测试完成");
    0
}
```

### 在rCore中的使用方法

1. **将文件添加到用户程序目录**:
   ```bash
   # 复制到rCore用户程序目录
   cp exercise_c_sleep.c ../rCore-Tutorial-v3/user/src/bin/exercise_c.rs
   ```

2. **修改文件扩展名和导入**:
   ```rust
   // 在 exercise_c.rs 中
   #![no_std]
   #![no_main]

   #[macro_use]
   extern crate user_lib;

   use user_lib::{sleep, *};

   #[no_mangle]
   pub fn main() -> i32 {
       println!("开始睡眠5秒...");
       sleep(5000);  // 注意：参数可能是毫秒
       println!("醒来！");
       0
   }
   ```

3. **编译并运行**:
   ```bash
   cd ../rCore-Tutorial-v3/os
   make run
   ```

### sleep系统调用说明

在rCore Tutorial的不同章节中，sleep的实现不同：

- **ch1分支**: 可能尚未实现sleep系统调用
- **ch2分支**: 批处理系统，可能实现了简单的延时
- **ch3分支**: 协作式调度，实现了sleep系统调用
- **ch4+分支**: 时钟中断驱动的sleep

### 如果ch1没有sleep实现

可以自己实现一个简单的忙等待版本：

```rust
#![no_std]
#![no_main]

#[macro_use]
extern crate user_lib;

use user_lib::*;

// 简单的延时循环
fn simple_delay(seconds: usize) {
    // 假设CPU频率约为某值，通过循环计数实现延时
    let loops_per_second = 1000000; // 需要根据实际情况调整

    for _ in 0..(seconds * loops_per_second) {
        unsafe { asm!("nop"); }  // 空操作
    }
}

#[no_mangle]
pub fn main() -> i32 {
    println!("Exercise C: Simple delay (busy wait)");
    println!("====================================");

    println!("开始延时5秒（忙等待）...");

    simple_delay(5);

    println!("延时结束！");

    0
}
```
