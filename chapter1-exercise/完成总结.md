### ✅ 练习C：rCore sleep系统调用

**状态**: ✅ 已完成代码编写并成功集成到rCore项目

**实现位置**:
- 概念代码: [exercise_c_sleep.c](exercise_c_sleep.c)
- rCore实现: `rCore-Tutorial-v3/user/src/bin/exercise_c.rs` ✅

**代码功能**:
```rust
#![no_std]
#![no_main]

#[macro_use]
extern crate user_lib;

use user_lib::{get_time, sleep};

#[unsafe(no_mangle)]
pub fn main() -> i32 {
    println!("Exercise C: Sleep System Call Test");

    let start_time = get_time();
    println!("准备睡眠 5000 ms (5秒)...");

    sleep(5000);  // 使用sleep系统调用

    let end_time = get_time();
    let elapsed = end_time - start_time;

    println!("实际睡眠时间: {} ms", elapsed);
    println!("Test Passed!");

    0
}
```

**测试方法**:
```bash
cd /path/to/rCore-Tutorial-v3/os
make run

# 在rCore shell中
>> exercise_c
```
